# SCB Config Repository

Centralized configuration repository for the **Spring Cloud Bus (SCB) demo**.  
This repository contains global and service-specific configuration files used by the `scb-config-server`.

## Structure

- **`application.yml`**  
  Contains global/shared configuration.

- **`campaign-service.yml`**  
  Contains configuration specific to the Campaign Service.


## Usage

scb-config-server will use this repo as file config.
